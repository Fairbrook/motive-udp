# Motive UDP

This is the markers sample project from Optitrack Motive API modified to send the position and attitude parameters through a UDP connection

To use this, open the solution in Visual Studio (2026) and modify the IP in the 461 line to match the host running the [Motive ROS](https://github.com/fairbrook/motive-ros)
project and hit run

To create a rigid body from the markers seen by the cameras or reset the current rigid body press K with the terminal focused
